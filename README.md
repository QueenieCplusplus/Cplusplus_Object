# Cplusplus_Object
在 C++ 中，物件是？

物件是一塊（內含某型別數值）的記憶體 memory，其中型別可以是自定義 self-defined 也可以是標準函式庫的類別 class 和型別 Types，而變數是一個具名物件 object，其內含的數值 value ，就是依據型別來解釋的位元 bite。

# Constant

    具有數值 immutable 不可變的概念，意思即『保證不會變更數值』。
